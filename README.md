\# Personal Finance Dashboard (Excel Project)



\## 📌 Objective

This project analyzes personal income and expenses to understand spending patterns and savings behavior.



\## 🛠 Tools Used

\- Microsoft Excel  

\- Pivot Tables  

\- Charts \& Dashboard  



\## 📊 Dataset

A self-created dataset containing 45+ financial transactions including income and expenses across different categories.



\## 📈 Analysis Performed

\- Category-wise spending analysis  

\- Income vs Expense comparison  

\- Monthly trend analysis  



\## 📊 Dashboard Features

\- Total Income, Expense \& Savings  

\- Category-wise spending (Pie Chart)  

\- Monthly trend (Line Chart)  

\- Interactive slicers (Month \& Category)  



\## 🧠 Key Insights

\- Food category contributes the highest share of expenses  

\- Total income is higher than expenses, indicating savings  

\- Spending is consistent across months  

\- High spending on shopping and entertainment
